# FCM Create Helper

用于在 Figma 里批量复制 `FCM模版`，并按 `CSV + 序号图片` 生成 FCM 草稿。

## 第一版能力

- 读取 CSV 中的 `id / song / artist`
- 按图片文件名中的序号匹配人物图
- 批量复制模板
- 自动替换 `歌名`、`歌手名`、`人物`
- 自动给 `背景`、`大圆`、`小圆` 生成一版初始渐变
- 按网格排布生成结果
- 上传 `webp` 等格式时会先转成 `PNG` 再写入 Figma
- 歌名单行时会尽量贴近原双行标题的底边，保持和歌手名的距离更接近模板
- 歌手名会尽量和 `Hires标志` 对齐，歌名则按模板原始间距回推
- 人物图导入时会自动加一点安全边距，避免默认看起来太大
- 可选开启“试验版本地去背”，先做人像分割后再导入模板
- 如果导入图片本身已经带透明背景，插件会自动跳过本地去背，适合透明 PNG 和普通原图混用

## 模板要求

在 Figma 模板中保留这些图层名：

- `人物`
- `歌手名`
- `歌名`
- `大圆`
- `小圆`
- `背景`

模板根节点建议命名为 `FCM模版`。

## CSV 示例

```csv
id,song,artist
001,Il cielo in una stanza,Gino Paoli
002,BLITZ,Niko Pandetta
003,Giovani Re,Sfera Ebbasta
004,Scrivendo,nayt
005,Bullet Ballet,Kid Yugi
```

图片文件名示例：

```text
001.jpg
002.jpg
003.jpg
004.jpg
005.jpg
```

## 使用方式

1. 在 Figma 里选中一个 `FCM模版`，或者确保当前页存在同名模板。
2. 导入 CSV。
3. 导入序号图片。
4. 如果需要，勾选“试验版本地去背”。
5. 点击“生成草稿”。
6. 插件会批量生成匹配成功的草稿，你再手动微调抠图、人物位置和颜色。

## 当前限制

- 默认会直接替换 `人物` 图层图片；若你勾选“试验版本地去背”，插件会先做人像分割再导入。
- 初始配色是根据图片局部颜色自动生成的草稿色，不保证最终审美结果。
- 若模板中某些目标图层缺失，插件会继续生成，但会给出提醒。
- Figma 插件 API 写入图片时稳定支持 `PNG / JPEG / GIF`，所以插件会把其它浏览器可解码格式先转成 `PNG` 后再导入。
- 本地去背是试验功能，第一次启用时需要联网下载 MediaPipe 模型，复杂头发和边缘细节仍可能需要你手工修一下。
